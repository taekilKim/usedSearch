# 중고 가격비교 통합 검색

중고 물건을 검색하면 여러 플랫폼의 가격을 한 번에 비교할 수 있는 Chrome 확장 프로그램입니다.

## 지원 플랫폼

- 번개장터 (bunjang.co.kr)
- 중고나라 (joongna.com)
- 당근마켓 (daangn.com)
- 알라딘 중고샵 (aladin.co.kr)
- 네이버 쇼핑 (shopping.naver.com)
- 이베이 (ebay.com)

## 주요 기능

- **통합 검색**: 한 번의 검색으로 여러 중고거래 플랫폼을 동시에 검색
- **가격 비교**: 모든 결과를 가격순으로 정렬하여 표시
- **가격 통계**: 최저가, 최고가, 평균가를 자동으로 계산
- **바로가기**: 각 상품의 상세 페이지로 바로 이동할 수 있는 링크 제공

## 설치 방법

1. 이 저장소를 클론하거나 다운로드합니다
2. Chrome 브라우저에서 `chrome://extensions/` 페이지를 엽니다
3. 우측 상단의 "개발자 모드"를 활성화합니다
4. "압축해제된 확장 프로그램을 로드합니다" 버튼을 클릭합니다
5. 다운로드한 폴더를 선택합니다

## 사용 방법

1. Chrome 툴바의 확장 프로그램 아이콘을 클릭합니다
2. 검색창에 찾고 싶은 중고 물건의 이름을 입력합니다 (예: "아이폰 13", "맥북 에어 2019")
3. "동시 검색" 버튼을 클릭합니다
4. 각 플랫폼의 검색 결과 페이지가 새 탭으로 열리고, 자동으로 데이터가 수집됩니다
5. 팝업에서 모든 플랫폼의 검색 결과를 가격순으로 확인할 수 있습니다

## 기술 스택

- Manifest V3
- Vanilla JavaScript
- Chrome Extensions API
- Content Scripts
